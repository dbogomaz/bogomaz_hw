Выполнение задания № 1
1. Создаем папку для задания .../task_1
2. Создаем файл my_commands.txt для ведения дневника исползуемых комманд
3. Создаем удаленный репозиторий на GitHub
    bogomaz_hw
4. Связываем локальный и удаленные репозитории 
    git clone https://github.com/dbogomaz/bogomaz_hw.git
5. Создаем файл task1.cpp, пишем туда код Hello word
6. Добавяем этот файл в список отслеживаемых и выкладываеме на сервер
    git add .
    git commit -m 'Hello word !'
    git push
7. Изменяем сообщение в файле на "Hello git" и выкладываем в удаленный репозиторий
    git add .
    git commit -m 'Hello git !'
    git push
